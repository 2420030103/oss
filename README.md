# IPC Chat Application (Java)

This project demonstrates Inter-Process Communication (IPC) using Java Socket Programming.

## Features
- Client-Server chat
- Two-way communication
- Runs on terminal

## How to Run

### Compile
javac Server.java
javac Client.java

### Run
Open two terminals:

Terminal 1:
java Server

Terminal 2:
java Client

## Output
Client and Server can send messages to each other in real-time.

## Concepts Used
- Socket Programming
- IPC
- Streams in Java
